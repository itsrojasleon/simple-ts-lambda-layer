# TS lambda layer

A simple example using lambda layers with NodeJS (Typescript).

## Install dependencies

```shell
# If using pnpm
cd services && pnpm i

# If using npm
cd services && npm i
```

## Run project locally

```shell
# If using pnpm
cd service && pnpm run dev

# If using npm
cd service && npm run dev
```
